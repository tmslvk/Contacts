Contacts 

Приложение для хранения контактов. Есть функции добавления, удаления, редактирования, поиска контактов.

Разработано с использованием .Net Core 8 и Vue.js 3.
Использовался EFCore. Для бд был выбран MS SQL Server.

Для корректной работы приложения рекомендуется настроить appsettings.json. Пример appsettings.Example.json.
А также воспользоваться генератором контактов для быстрой проверки работоспособности приложения. Закоменченный код в Program.cs и класс DbSeeder.
